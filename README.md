# flowerclassifier

## Setup

1. Clone the repo
2. Train the model: python train.py (from github.com/DenkoBetona/nm_flowermodel)
3. Export to ExecuTorch: python export.py (from github.com/DenkoBetona/nm_flowermodel)  
4. Copy the generated flowers.pte to app/src/main/assets/
5. Open in Android Studio and run
